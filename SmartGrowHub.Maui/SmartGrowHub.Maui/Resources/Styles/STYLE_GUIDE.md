# Planned refactoring of label style names
## Label styles
- **Sizes:**  
  `XSmall` (12px), `Small` (14px), `Medium` (16px), `Large` (18px), `XLarge` (24px).
- **Weight:**  
  `Regular`, `Medium`, `Semibold`, `Bold`.
- **Color:**  
  `Primary` (основной), `Secondary` (второстепенный), `Gray`, `Accent` (акцентный).
